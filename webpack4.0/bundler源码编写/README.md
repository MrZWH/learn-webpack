
cli-highlight 高亮显示代码的工具： 
```shell
npm i cli-highlight -g
```

运行： 
```
node bundler.js | highlight
```

借助`@babel/parser`获取文件中的依赖：
```shell
npm i @babel/parser -S
```

`@babel/traverse`快速获取使用 import的语句
```
npm i @babel/traverse -S
```

使用 babel 编译代码：
```
npm i @babel/core -S
```

preset:
```
npm i @babel/preset-env -S
```
